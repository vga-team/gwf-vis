# GWF-VIS

With the earth experiencing a drammatically changing climate, scientists are relying on hydrological models to understand and cope up with extreme events such as droughts and floods. The GWF-Vis library provides a set of APIs to create an interactive visual analytics dashboard to visualize geospatial data such as the ones produced by hydrological models. The GWF-Vis library ([Wiki](https://github.com/vga-team/gwf-vis_lib/wiki)) can be installed through a terminal or Jupyter Notebook. Interested users may refer to ([VGA](https://github.com/vga-team)) for development details.

Check more on our [Website](https://vga-team.github.io/gwf-vis_site/).
